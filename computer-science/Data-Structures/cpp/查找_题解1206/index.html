# 3.A-B 数对（map 库的引入）

开始写了一个暴力查找的优化，但室友提到用 map 库，很爽，花了一点时间学了一下

map 优势如下：

1.map 类似于数组，比如数组 s [1], 但 map 可以让这里的数字 1 为任何索引，比如 s ['a'],s [10000],s [-1], 等等

2.map 函数本质是二叉树，红黑树，查找效率为 log2n，很方便

3. 因此，在建立好索引后，我们可以采用第 2 题的思想来做
